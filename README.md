# Estoque-java
# Sistema de Controle de Estoque (Java)

Aplicação desenvolvida em Java com foco em lógica de programação e Programação Orientada a Objetos (POO).

 Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Atualização de quantidade
- Remoção de produtos

 Conceitos aplicados
- Programação Orientada a Objetos (POO)
- ArrayList
- Estruturas condicionais
- Entrada de dados com Scanner

 Tecnologias
- Java
